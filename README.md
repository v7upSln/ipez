A lightweight command-line system information tool for macOS and iOS that displays hardware and software details.

## Features

- Displays device model and hardware information
- Shows iOS/macOS version and kernel details
- Reports CPU and GPU specifications
- Battery percentage monitoring
- System uptime tracking
- RAM and storage usage statistics
- Clean, colorful terminal output

## Installation

Add my repository and install via package manager. [my repo](https://v7upsln.github.io)

### Usage

```bash
echo
```

The tool will display comprehensive system information including device model, OS version, CPU, GPU, battery status, uptime, and storage details.

## System Information Displayed

- Device model
- iOS/macOS version
- Kernel version
- CPU brand and architecture
- GPU name
- Battery percentage
- System uptime
- RAM capacity
- Storage capacity and availability
